# Diffusion和薛定谔桥的对比

相比于Diffusion, DSB主要有以下两个优点：

- DSB计算的是两个分布之间的最优传输（Optimal Transport，OT）；

- DSB可以计算任意两个可采样分布之间的OT，而相比之下Diffusion只能从已知分布（如高斯噪声）开始生成数据。
